# Fake News & Misinformation Detector (PS03) 🛡️
**Team:** HACKTIVATE SYNDICATE  
**Event:** The Sandbox IT Fest 2026 - SVPCET Nagpur

## 🚀 Overview
An advanced, full-stack AI platform designed to combat the spread of digital misinformation. By analyzing raw text or direct news URLs, the tool provides a deep-dive audit into the credibility, emotional manipulation, and political bias of online content.

## ✨ Key Features
- **Real-Time Web Scraping:** Automatically extracts clean article content from news URLs, bypassing ads and navigation clutter.
- **AI-Powered Analysis:** Leverages the **Llama 3.3-70B** model (via Groq) for lightning-fast inference.
- **Credibility Scoring:** Generates a 0-100 score based on factual consistency and source reliability.
- **Bias & Manipulation Detection:** Visualizes political leaning (Left to Right) and flags emotionally manipulative language.
- **Interactive Dashboard:** High-performance React UI featuring a dark-mode analysis report and local session history.

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Tailwind CSS, Lucide React
- **Backend:** Node.js, Express
- **AI/ML:** Groq API (Llama 3.3)
- **Scraping Engine:** Cheerio, Axios

## ⚙️ Installation & Setup
1. Clone the repository: `git clone [repo-link]`
2. **Backend:**
   - `cd backend`
   - `npm install`
   - Create a `.env` file with your `GROQ_API_KEY`
   - `node server.js`
3. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
